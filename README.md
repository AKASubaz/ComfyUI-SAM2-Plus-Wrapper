# ComfyUI-SAM2-Plus-Wrapper

A ComfyUI custom node wrapper for [SAM2-Plus](https://github.com/MCG-NJU/SAM2-Plus) (tracking anything at any granularity).

## Installation

    *   Clone this repo into `ComfyUI/custom_nodes/`
    *   Run `pip install -r requirements.txt`

## Model Setup
Download the SAM2++ checkpoints and place them in `ComfyUI/models/sam2/`.
https://huggingface.co/MCG-NJU/SAM2-Plus/resolve/main/checkpoint_phase123.pt
